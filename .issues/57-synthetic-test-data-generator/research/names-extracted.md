## Slawische Frauennamen-Endungs-Logik (für Generator)

Wenn weibliche Person in Cluster `osteuropa` oder `ex-yu`:

| Nachname endet auf | Frauen-Form |
|---|---|
| `-ski` (PL) | `-ska` |
| `-cki` (PL) | `-cka` |
| `-cký` (CZ/SK) | `-cká` |
| `-ský` (CZ/SK) | `-ská` |
| `-ý` (CZ generic) | append `-á` (z.B. Novák→Nováková for women — special case: Novák stem + ová) |
| Default CZ/SK | append `-ová` |
| `-ić` (Serbian/Croatian) | KEINE Änderung (gleich für beide) |
| Hungarian (no gender suffix) | KEINE Änderung |
| Romanian | KEINE Änderung |

Beispiele:
- Kowalski (m) → Kowalska (f)
- Novák (m) → Nováková (f)
- Petrović (m) → Petrović (f)
- Nagy (m/f, ungarisch) → Nagy (f)
